This is a straightforward console-based application built in Python designed to generate simple
retail bills. It allows a user to input multiple items, their quantities, and unit prices, and then

formatted receipt.

üöÄ How to Run
Prerequisites
You only need a working installation of Python 3.
Running the Application
1. Save the Code: Ensure the Python code is saved in a file named billing_system.py.
2. Open Terminal: Navigate to the directory where you saved the file.
3. Execute the Script: Run the program using the Python interpreter:
python billing_system.py
Usage
1. When prompted, enter the name of the item.
2. Enter the quantity (must be a whole number greater than 0).
3. Enter the unit price (must be a positive number).
4. Repeat steps 1-3 for every item you wish to add to the bill.
5. To finish adding items, type done when prompted for the item name.
6. The program will then display the complete, formatted bill.
